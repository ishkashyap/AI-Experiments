# Dejure Law Office - Admin Panel User Manual

## How to Access the Admin Panel

1. Open your web browser (Chrome, Firefox, Safari)
2. Go to: `http://localhost:3000/admin.html`
3. Login with:
   - Username: **admin**
   - Password: **admin123**

---

## Navigating the Admin Panel

After logging in, you will see tabs at the top:
- 🏠 **Home Page** - Edit the main landing page
- ℹ️ **About Page** - Edit the About page content  
- 📝 **Content** - Manage website content
- 🏢 **Offices** - Manage office locations
- 📢 **Announcements** - Post notices
- 👣 **Footer** - Edit footer contact info
- 📚 **Blogs** - Manage blog posts
- 📅 **Bookings** - View client bookings

---

## How to Edit Home Page

Click on **Home Page** tab. You will see these sections:

### 1. Hero Section
- **Hero Tagline**: Small text above the main title (e.g., "Excellence in Legal Advocacy")
- **Hero Title Line 1**: Main heading (e.g., "Clarity. Strategy.")
- **Hero Title Line 2**: Second line (e.g., "Resolution.")
- **Hero Subtitle**: Description text below the title
- **Stat 1, 2, 3**: Numbers shown below (e.g., "25+", "500+", "98%")

**To edit any field:**
1. Click inside the text box
2. Type your new text
3. Click the **Save** button next to that field
4. You will see a green "Saved successfully" message
5. Refresh your website to see the changes

### 2. "The Firm" Image
- Enter an image URL or click "Choose File" to upload a new image
- Click **Save Image**

### 3. "The Firm" Section
- Edit the tagline, title, paragraph, and bullet points
- Each field has its own Save button

### 4. Practice Areas Section
- Edit the section title and all 4 services
- Each title and description has its own Save button

---

## How to Edit Footer

Click on **Footer** tab. You can edit:
- **Address** - Office address
- **Phone** - Contact phone number  
- **Email** - Contact email
- **Working Hours** - Business hours
- **Description** - Footer text about the firm

Each field has its own Save button.

---

## How to Edit About Page

Click on **About Page** tab:
- Edit the Story Title and paragraphs
- Edit Mission text
- Edit Expertise cards (4 cards)
- Manage Team Members (Add/Edit/Delete)

---

## Important Tips

✅ **Individual Updates**: You can change just ONE field without affecting others!
- Example: Changing the phone number won't reset the address or logo

✅ **Refresh Required**: After saving, refresh your website to see changes

✅ **No Data Loss**: Leaving a field blank won't erase existing content

✅ **Image Upload**: Use PNG or JPG images for best results

---

## Troubleshooting

**"Failed to fetch" error:**
- Check that the server is running
- Refresh the admin page and try again

**Changes not showing:**
- Make sure to click the Save button for each field
- Refresh the website after saving

**Can't login:**
- Use username: admin
- Use password: admin123
- Make sure server is running
